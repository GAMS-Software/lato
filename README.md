# Lato
Basic engine for all Lato projects.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "lato"
```

And then execute:
```bash
$ bundle
$ rails lato:install:migrations
$ rails db:migrate
```
